Zynthian Customs Document

This is the module to create customs document for Zynthian in Odoo 9.0

This module allow you to create customs pdf document.

Zynthian customs document in Sale Order

    Customs document is made with prefixed images that we use like watermark in order to create Custom Report. In order to change preferences is necessary to modify this images that will be find in /src/img/ directory.
    About sender and receiver information is fill in customs_template.xml, the position is handmade so if the images are changed, will be necessary callculate the positions again.
    Report will be locate in Sale Order Print action and you can print more than one sale order at any time.
